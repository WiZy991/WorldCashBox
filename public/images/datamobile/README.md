# Изображения DataMobile

## Как добавить изображения

### Вариант 1: Локальные файлы (рекомендуется)

1. Скачайте изображения из Яндекс.Диска
2. Переименуйте их согласно списку ниже
3. Поместите файлы в эту папку (`public/images/datamobile/`)

**Необходимые файлы:**
- `datamobile-main.jpg` - главный баннер DataMobile (для hero секции)
- `dm-delivery-pro.jpg` - баннер DM.Доставка Pro
- `dm-invent.jpg` - баннер DM.Invent
- `dm-toir.jpg` - баннер DM.ТОИР
- `dm-pricechecker.jpg` - баннер DM.Прайсчекер

### Вариант 2: Прямые ссылки из Яндекс.Диска

Если хотите использовать прямые ссылки на изображения из Яндекс.Диска:

1. Откройте файл изображения в Яндекс.Диске
2. Нажмите "Поделиться" → "Получить ссылку"
3. Скопируйте прямую ссылку (она должна начинаться с `https://disk.yandex.ru/i/...`)
4. Откройте файл `app/datamobile/page.tsx`
5. Замените пути в поле `image` на прямые ссылки

**Пример:**
```typescript
image: 'https://disk.yandex.ru/i/abc123def456', // вместо '/images/datamobile/dm-delivery-pro.jpg'
```

## Форматы изображений

Рекомендуемые форматы:
- JPG/JPEG - для фотографий и баннеров
- PNG - для изображений с прозрачностью
- WebP - для оптимизированных изображений

Рекомендуемый размер:
- Главный баннер: 1200x600px или больше
- Баннеры продуктов: 800x600px или больше
